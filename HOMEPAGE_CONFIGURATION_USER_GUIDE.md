## üö® Common Issues & Solutions

### Issue 1: "Cannot publish"
**Cause:** Config validation failed  
**Solution:** Check all required fields filled

### Issue 2: "Changes not showing"
**Cause:** Cache not cleared  
**Solution:** Wait 1 hour or trigger revalidation

### Issue 3: "Section not displaying"
**Cause:** Section disabled or hidden  
**Solution:** Check enabled toggle and visibility rules

### Issue 4: "Images not loading"
**Cause:** Invalid image URL  
**Solution:** Use valid HTTPS URL or upload file

### Issue 5: "Preview shows old version"
**Cause:** Browser cache  
**Solution:** Hard refresh (Ctrl+F5)
